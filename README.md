## 🎯 학습 목표 🎯

논리 회로, 하드웨어 면에서의 컴퓨터 구조 등에 대한 내용을 이해하며
기계어와 어셈블리어 프로그래밍을 통해 컴퓨터 내부 동작 방식을 이해하도록 한다.
소프트웨어 프로그래밍 시 하드웨어 구조를 고려하여 더욱 효율적인 프로그래밍을 하도록 한다.
